List Tag

HTML list <li> tag is used if you want to enter the contents in the listed order. There are two types of lists.

  • Ordered list < ol >
  • Unordered list < ul >

HTML lists allow web developers to group a set of related items in lists.

Example

An Ordered HTML list:

  1. item
  2. item
  3. item
  4. item

An Unordered list:

  • First item
  • Second item
  • Third item
  • Fourth item
